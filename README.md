## 📖 The Vision
**Project Bridge** is an open-source, AI-driven intranet platform created to solve a complex socio-medical challenge: How do we protect elderly individuals with neurodegenerative conditions (like Frontotemporal Dementia, FTD) from internet scams, without isolating them and stripping away their dignity?

Instead of simply disconnecting the router, this project creates a **"Logical Intranet"**. It uses a multi-modal AI Agent to filter out malicious content and redirect the user's attention toward their lifelong professional expertise—in this case, Mechanical Engineering. It transforms a vulnerable patient into a respected "Chief Technical Reviewer."

## ✨ Key Features

* 🛡️ **Semantic Firewall (语义防火墙):** Unlike traditional keyword blockers, our LLM-based agent analyzes the *intent* of user searches. If a user searches for scam-related or obsessive topics, the Agent gently redirects the conversation back to academic engineering topics using contextual association.
* 👨‍🏫 **Reverse-Mentorship UI (反向师徒界面):** The interface is designed not for a "patient," but for a "Professor." It features a formal, academic dashboard where the elderly user reviews and approves engineering assignments submitted by the younger generation.
* 📐 **Cross-Standard Engineering Translation:** Built-in AI bridges the gap between different international engineering standards (e.g., ISO First-Angle vs. ANSI Third-Angle projection), facilitating seamless cross-cultural and cross-generational technical communication.
* 🔒 **Kiosk-Mode Deployment:** Designed to run as a Progressive Web App (PWA) within an iOS Guided Access environment, ensuring the user remains safely within the platform's ecosystem.

## 🛠️ Tech Stack

* **Frontend & Backend:** Python, [Streamlit](https://streamlit.io/)
* **AI Integration:** OpenAI GPT-4o API (for semantic routing and vision-based drawing review)
* **Deployment:** Streamlit Community Cloud / Vercel
